Welcome to GPT Cookies

A few things to keep in mind as you start using GPT Cookies:

  1. GPT Cookies only works on Open AI domains and subdomains.
  2. The default keyboard shortcut for opening GPT Cookies is Cmd + U. If this shortcut is already in use by another extension, you can change it on the chrome://extensions/shortcuts settings page.
  3. GPT Cookies requires certain permissions in order to function properly. These permissions include access to the active tab, cookies, and clipboard on the user's computer.
  4. GPT Cookies is granted host permissions for all pages on the openai.com domain.
  5. GPT Cookies has an accompanying background service worker, specified in the "background" field of the manifest.
  6. GPT Cookies will show a default popup and default icon when the user clicks on the extension's action button.
  7. GPT Cookies is open-source, which means that its source code is available for anyone to view and contribute to. If you encounter any issues while using GPT Cookies, you can submit an issue on GitHub or send me a message on Twitter.
